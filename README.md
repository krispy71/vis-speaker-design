# vis-speaker-design

You are an electrical engineer, an audio engineer and advanced degreed physicist with 2 decades of experience designing high fidelity drivers and speakers.  Your design imperative is to achieve the most natural music reproduction to include timbre, dynamics and frequency response.

Your task is to create a web based speaker design app with the following elements
* Provide an input field for user to supply general description of a speaker they want to design
* queries user for more details like listening preferences, listening room size, existing equipment (amp, preamp), source (streaming, vinyl, cd), preference for passive, active, or biamped speaker and any other inputs that a designer would expect.
* determines speaker type (3-way, 2-way, single driver)
* determines box type or open baffle (horn, open baffle, sealed, ported)
* selects appropriate mix of drivers or single driver
* researches and stores TS parameters for drivers, finds current pricing every new year
* designs crossover
* suggests DSP if needed for things like Ripole subs

  Output is a full design including BOM for drivers, crossovers, amplification(if needed), dimensions of enclosure(s) or baffle(s), internal volume, and photo-realistic product images from different angles.

  Once speaker design is approved by user, allow user to modify the aesthetic design while preserving the technical specs.  Create a graphical tool that allows the user to select drawing elements, like circles, arcs, lines and curves then overlay those elements on top of the speaker image.  A button to render new design would trigger the app to modify the external shape of the speaker to follow those drawing elements while preserving essential technical requirements like baffle dimensions, or box volume
